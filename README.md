# Phishing URL Detection using Machine Learning

This project detects phishing websites using machine learning techniques.
It uses URL-based features and Logistic Regression to classify URLs as phishing or legitimate.

Files
- phishing_detection.ipynb – Jupyter notebook with implementation
- phishing.csv – Dataset used for training and testing

Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

Output
The model predicts whether a given URL is phishing or legitimate.
